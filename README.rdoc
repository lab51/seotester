SEOtester - online at http://seotest.carted.pl.

Written for my friends - they are creating a site at carted.pl and they needed a tool to do a quick SEO test. 

This tool checks things like:

- title length
- number of H1 tags
- redirections status
- page speed
- page rank 
- and other things related to SEO 

It does not use a database. In the end SEOtester prints score and recommendations.
